# Expo 2025 パビリオン予約自動化スクリプト

大阪・関西万博2025のパビリオン予約を自動化するNode.jsスクリプトです。

## 📋 概要

このスクリプトは、Puppeteerを使用してExpo 2025の公式チケット予約サイトを自動操作し、利用可能なパビリオンの予約を効率的に行います。

## 🚀 機能

- **自動パビリオン検索**: 利用可能なパビリオンを自動的に検索
- **除外フィルター**: 指定したパビリオンを除外して検索
- **時間制限**: 指定した時間帯（デフォルト：19時まで）内の予約のみ対象
- **予約可能性チェック**: ◯（空きあり）、△（残りわずか）のパビリオンを自動検出
- **エラーハンドリング**: 予約失敗時の自動リトライ機能
- **ユーザー操作待機**: 必要な場面でのユーザー入力待機

## 📦 必要な環境

- Node.js (v14以上推奨)
- Google Chrome ブラウザ
- インターネット接続

## 🛠️ インストール

1. リポジトリをクローンまたはダウンロード
```bash
git clone <repository-url>
cd expo2025-booking-automation
```

2. 依存関係をインストール
```bash
npm install puppeteer
```

## ⚙️ 設定

### 除外パビリオンの設定
`EXCLUDED_PAVILIONS` 配列で除外するパビリオンを設定できます：

```javascript
const EXCLUDED_PAVILIONS = [
  '車いす',
  '障がい者',
  'バリアフリー',
  // 他の除外したいパビリオン名
];
```

### 予約時間制限の設定
`RESERVATION_TIME_LIMIT` で予約を行う時間帯を制限できます：

```javascript
const RESERVATION_TIME_LIMIT = 19; // 19時までの予約のみ対象
```

## 🎯 使用方法

1. スクリプトを実行
```bash
node index.js
```

2. ブラウザが自動的に開かれ、Expo 2025のチケット予約サイトにアクセスします

3. 手動でログインを行ってください

4. パビリオン検索ページに移動したら、ターミナルでEnterキーを押してください

5. スクリプトが自動的に以下を実行します：
   - パビリオンの検索
   - 利用可能な時間帯の確認
   - 予約の実行
   - 結果の表示

## 🔧 カスタマイズ

### 待機時間の調整
`randomDelay()` 関数で待機時間を調整できます：

```javascript
await randomDelay(1000, 3000); // 1-3秒のランダム待機
```

### ブラウザ設定
Puppeteerのlaunchオプションで設定を変更できます：

```javascript
const browser = await puppeteer.launch({ 
  headless: false,  // ヘッドレスモードの切り替え
  defaultViewport: null,
  // その他のオプション
});
```

## 🤝 トラブルシューティング

### よくある問題

1. **ブラウザが開かない**
   - Google Chromeがインストールされているか確認
   - Puppeteerが正しくインストールされているか確認

2. **セレクターエラー**
   - サイトのUI変更により発生する可能性があります
   - 該当のセレクターを最新のものに更新してください

3. **予約に失敗する**
   - 時間制限の設定を確認
   - 除外パビリオンの設定を確認
   - インターネット接続を確認
